Nevent Newsletter Widget Example

About this example

This is a basic example of the @nevent/subscriptions package. The widget loads configuration from the Nevent API and renders a GDPR-compliant subscription form.

Features demonstrated:

  • Automatic configuration loading from server
  • Custom fonts (Google Fonts & custom fonts)
  • Form validation with user-friendly errors
  • GDPR consent checkbox
  • Responsive layout
  • Success/error states
